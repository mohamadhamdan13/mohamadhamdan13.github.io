Admin-närvaro

Detta projekt genomfördes som ett skolprojekt där Node v16.17.1 och Python 3.10.6 användes. Huvudsyftet med projektet var att skapa tillgänglighetsindikatorer för lärare. Genom att använda närvarolådor hade lärarna möjlighet att signalera om de var tillgängliga eller inte genom att välja alternativen "tillgänglig" eller "inte tillgänglig". Busskylt

Vi på TE4 genomförde ett skolprojekt där vi skapade en busskylt för elever och personal på skolan. Denna skylt presenterade information om de busshållplatser som var närmast skolan tillsammans med de två kommande busstiderna. För att integrera realtidsinformation använde vi API:er. De huvudsakliga teknologierna och språken som användes för att implementera projektet inkluderade HTML5, CSS3 och JavaScript.
